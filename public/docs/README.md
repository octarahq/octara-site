# Octara Documentation

Welcome to the technical documentation for Octara, a secure Forge infrastructure and OAuth2 provider.

## Key Endpoints

- **Base URL (API)**: `https://octara.xyz/api/v1`
- **Authentication (OAuth2)**:
  - **Authorize**: `https://octara.xyz/oauth/authorize`
  - **Token Exchange**: `https://octara.xyz/api/oauth/token`

## Available Resources

- **OpenAPI Specification**: Detailed API documentation in YAML format.
- **OAuth2 Guide**: Step-by-step implementation guide for developers.

For more information, visit the [Octara Dashboard](https://octara.xyz/account/apps).
